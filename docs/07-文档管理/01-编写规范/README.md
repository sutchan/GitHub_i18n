# 编写规范

本目录包含GitHub中文翻译插件文档的编写规范和要求，指导文档编写者如何编写符合规范的文档。

## 目录内容

- 目前目录下暂无具体文件，编写规范内容将在后续添加

## 编写规范概述

文档编写规范包含以下几个方面：

1. 文档结构：文档的目录结构和组织方式
2. 内容要求：文档内容的准确性、完整性和清晰度
3. 格式规范：文档的排版、字体、颜色等格式要求
4. 语言规范：文档使用的语言、术语和表达方式
5. 图片和链接：图片和链接的使用规范
