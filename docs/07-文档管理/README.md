# 文档管理

本目录包含GitHub中文翻译插件的文档管理规范和指南，介绍了项目文档的编写、审查和更新流程。

## 目录结构

- [01-编写规范](01-编写规范/)：包含文档编写的规范和要求
- [02-审查流程](02-审查流程/)：包含文档审查的流程和方法
- [03-更新指南](03-更新指南/)：包含文档更新的指南和要求

## 文档管理流程

1. 编写：按照编写规范编写文档
2. 审查：按照审查流程进行文档审查
3. 更新：按照更新指南更新文档
4. 归档：归档旧版本文档
