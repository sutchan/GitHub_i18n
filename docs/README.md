## 快速开始

如果您是第一次接触本项目，建议按以下顺序阅读文档：

1. 首先查看[项目概述](../02-项目概述/README.md)，了解项目的基本情况
2. 然后阅读[安装指南](../03-安装指南/README.md)，学习如何安装和配置插件
3. 接着阅读[功能模块](../04-功能模块/README.md)，了解插件的主要功能
4. 如果遇到问题，可以查看[常见问题](../07-常见问题/README.md)或[用户操作指南](../06-用户操作指南/README.md)
5. 如果您想参与开发，请参考[开发者文档](../07-开发者文档/README.md)
