# 常见问题

本目录包含GitHub中文翻译插件用户使用过程中可能遇到的问题及解决方案。

## 目录内容

- [常见问题指南.md](常见问题指南.md)：包含用户使用过程中常见问题的解答

## 如何使用

1. 浏览常见问题列表，查找与您遇到的问题相关的条目
2. 按照解决方案进行操作
3. 如问题仍未解决，请联系项目维护者

## 常见问题分类

- 安装问题：插件无法安装或安装后无法正常工作
- 翻译问题：翻译不准确或不完整
- 性能问题：插件导致页面加载缓慢或卡顿
- 兼容性问题：插件与其他扩展或浏览器版本不兼容
- 其他问题：无法归类到上述类别的问题
