# 用户指南

本目录包含GitHub中文翻译插件的用户指南，帮助用户了解如何使用该插件。

## 目录结构

- [01-基本使用](01-基本使用/)：包含插件的基本使用方法和操作指南
- [02-高级功能](02-高级功能/)：包含插件的高级配置和功能说明
- [03-常见问题](03-常见问题/)：包含用户使用过程中可能遇到的问题及解决方案

## 开始使用

1. 首先阅读[基本使用](01-基本使用/)目录下的用户操作指南
2. 如需自定义插件行为，请参考[高级功能](02-高级功能/)目录下的配置选项
3. 如遇到问题，请查看[常见问题](03-常见问题/)目录下的常见问题指南
