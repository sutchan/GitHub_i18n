# 基本使用

本目录包含GitHub中文翻译插件的基本使用方法和操作指南，帮助用户快速上手使用该插件。

## 目录内容

- [用户操作指南.md](用户操作指南.md)：详细介绍了插件的安装、配置和基本使用方法

## 开始使用

1. 首先确保您的浏览器已安装了支持用户脚本的扩展（如Tampermonkey、Violentmonkey等）
2. 安装GitHub中文翻译插件
3. 打开GitHub网站，插件将自动生效
4. 如需调整翻译设置，请点击浏览器扩展栏中的插件图标
