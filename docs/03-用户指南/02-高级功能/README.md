# 高级功能

本目录包含GitHub中文翻译插件的高级配置和功能说明，帮助用户自定义插件行为。

## 目录内容

- [配置选项.md](配置选项.md)：详细介绍了插件的各种配置选项和自定义方法

## 高级配置

插件提供了丰富的配置选项，允许用户根据自己的需求自定义翻译行为：

1. 翻译开关：可以开启或关闭特定页面或元素的翻译
2. 翻译语言：可以选择翻译的目标语言
3. 翻译质量：可以调整翻译的质量和准确性
4. 自定义词典：可以添加或修改翻译词典
5. 快捷键设置：可以自定义插件的快捷键
