# 版本历史

本目录包含GitHub中文翻译插件的版本更新记录，记录了每个版本的更新内容和变更说明。

## 目录内容

- [版本历史文档.md](版本历史文档.md)：详细记录了项目的版本更新内容

## 版本命名规范

项目采用语义化版本规范（SemVer），版本格式为：MAJOR.MINOR.PATCH

- MAJOR：不兼容的破坏性变更
- MINOR：向下兼容的新功能
- PATCH：向下兼容的问题修复

## 查看版本历史

1. 浏览[版本历史文档.md](版本历史文档.md)，了解每个版本的更新内容
2. 按照版本号从高到低排序，最新版本在最前面
3. 每个版本包含更新日期、版本号和更新内容
