## 📖 术语表

- **用户脚本 (Userscript)**: 一种可以在浏览器中运行的 JavaScript 脚本，通常通过用户脚本管理器安装
- **油猴 (Tampermonkey)**: 最流行的用户脚本管理器之一，支持多种浏览器
- **Violentmonkey**: 另一种常用的开源用户脚本管理器
- **MutationObserver**: 用于监听 DOM 变化的 Web API，是本项目实现自动翻译的核心技术
- **本地化 (Localization/i18n)**: 将软件界面和内容转换为特定语言和地区的过程
- **翻译词典**: 存储原文和翻译文本对应关系的数据结构
- **SPA (Single Page Application)**: 单页应用，如 GitHub 网站大量使用的前端技术
- **DOM (Document Object Model)**: 文档对象模型，网页的编程接口

## 🤝 贡献指南

欢迎通过以下方式为项目贡献：

1. **问题报告**: 提交 bug 报告或功能建议到项目 issue 页面
2. **代码贡献**: 提交代码改进或修复的 Pull Request
3. **翻译贡献**: 帮助完善翻译词典
4. **文档改进**: 撰写或改进项目文档
5. **社区推广**: 在社区中分享和推荐项目

详细的贡献指南将在 03-贡献指南.md 中提供。
