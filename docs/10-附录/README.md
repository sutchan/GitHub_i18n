## 术语表

- **用户脚本 (Userscript)**: 一种可以在浏览器中运行的 JavaScript 脚本，通常通过用户脚本管理器安装
- **油猴 (Tampermonkey)**: 最流行的用户脚本管理器之一，支持多种浏览器
- **MutationObserver**: 用于监听 DOM 变化的 Web API，是本项目实现自动翻译的核心技术
- **本地化 (Localization/i18n)**: 将软件界面和内容转换为特定语言和地区的过程
- **翻译词典**: 存储原文和翻译文本对应关系的数据结构
