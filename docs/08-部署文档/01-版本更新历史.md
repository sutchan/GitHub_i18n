# GitHub i18n 版本更新历史

## 📝 版本更新记录

### v1.8.62 - 2024-06-01

**主要更新：页面模式支持与性能优化**

#### 核心功能优化
- 实现了 `pageMode` 页面模式参数支持，使翻译引擎能够根据不同页面类型智能调整行为
- 优化了 `isImportantElement` 方法，添加了基于页面模式的特定判断逻辑
- 增强了 `shouldIgnoreElement` 方法，实现针对不同页面模式的定制化忽略规则
- 改进了 `isContentRelatedMutation` 方法，添加了页面模式特定的文本变化阈值
- 重构了 `detectImportantChanges` 方法，支持页面模式特定的性能配置和阈值

#### 性能优化
- 添加了基于页面模式的元素检测缓存机制，显著提高了重复检测的效率
- 实现了 `getTextChangeThreshold` 方法，为不同页面模式设置合理的文本变化阈值
- 完善了 `shouldSkipElementByPageMode` 方法，提供了更精细的元素过滤逻辑
- 针对 issues/pullRequests、wiki、search、codespaces 等特定页面模式优化了检测规则
- 优化了字符数据变化和子节点变化的处理逻辑，提高了翻译准确性

#### 代码质量改进
- 统一使用 `currentPageMode` 变量替代直接的 `pageMode` 参数，提高代码一致性
- 添加了页面模式存在性检查，增强了代码健壮性
- 优化了多处文本长度检查，使用页面模式特定的阈值配置
- 确保了所有核心方法都正确接收和使用页面模式参数

### v1.8.61 - 2024-05-25
