# 性能优化

本目录包含GitHub中文翻译插件的性能优化文档，介绍了项目的性能优化策略和实践。

## 目录内容

- [性能优化.md](性能优化.md)：详细介绍了项目的性能优化策略和实践

## 性能优化策略

1. DOM变化监听优化：采用高效的DOM监听机制，避免不必要的计算
2. 节流和防抖：对频繁触发的事件进行节流和防抖处理
3. 翻译触发条件优化：优化翻译触发条件，避免不必要的翻译
4. 翻译结果缓存：缓存翻译结果，避免重复翻译
5. 懒加载：对非关键资源进行懒加载
6. 内存管理：优化内存使用，避免内存泄漏
