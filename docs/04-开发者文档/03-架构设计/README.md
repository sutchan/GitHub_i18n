# 架构设计

本目录包含GitHub中文翻译插件的架构设计文档，帮助开发者了解项目的整体架构和功能模块。

## 目录内容

- [功能模块.md](功能模块.md)：详细介绍了项目的功能模块和架构设计

## 架构概述

GitHub中文翻译插件采用模块化架构设计，主要包含以下核心模块：

1. 入口模块：处理UserScript元数据和初始化
2. 配置模块：管理插件配置和配置界面
3. 翻译核心模块：执行翻译操作的核心逻辑
4. 页面监控模块：监控页面变化并触发翻译
5. 工具模块：提供各种工具函数
6. 错误处理模块：处理和记录错误
7. 版本管理模块：管理版本信息和检查更新
