# 贡献指南

本目录包含GitHub中文翻译插件的贡献指南，帮助开发者了解如何参与项目开发和贡献代码。

## 目录内容

- [贡献指南.md](贡献指南.md)：详细介绍了项目贡献的流程和规范

## 贡献流程

1. Fork项目仓库
2. 创建新的分支
3. 进行开发和修改
4. 提交代码并推送至远程仓库
5. 创建Pull Request
6. 等待代码审查和合并

## 贡献规范

- 代码风格：遵循项目现有的代码风格和命名约定
- 提交信息：清晰、简洁地描述提交的内容
- 测试：确保提交的代码通过所有测试
- 文档：更新相关文档以反映代码变更
