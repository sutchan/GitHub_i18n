# 开发者文档

本目录包含GitHub中文翻译插件的开发者文档，帮助开发者了解如何参与项目开发和贡献代码。

## 目录结构

- [01-贡献指南](01-贡献指南/)：包含项目贡献的流程和规范
- [02-API文档](02-API文档/)：包含项目的API接口说明
- [03-架构设计](03-架构设计/)：包含项目的架构设计和功能模块说明

## 开始贡献

1. 首先阅读[贡献指南](01-贡献指南/)，了解项目的贡献流程和规范
2. 查看[架构设计](03-架构设计/)，了解项目的整体架构和功能模块
3. 参考[API文档](02-API文档/)，了解项目的API接口
4. 克隆项目代码，开始开发
5. 提交代码前，确保通过所有测试

## 开发环境

- Node.js
- npm或yarn
- Git
