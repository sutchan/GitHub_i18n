# API文档

本目录包含GitHub中文翻译插件的API文档，帮助开发者了解项目的API接口和使用方法。

## 目录内容

- [API文档.md](API文档.md)：详细介绍了项目的API接口和使用方法

## API分类

- 翻译API：用于执行翻译操作的接口
- 配置API：用于管理插件配置的接口
- 事件API：用于处理插件事件的接口
- 工具API：提供各种工具函数的接口

## 使用方法

1. 导入所需的API模块
2. 调用相应的API函数
3. 处理返回结果
