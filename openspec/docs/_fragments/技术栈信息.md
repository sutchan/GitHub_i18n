## 核心技术栈

### 前端技术
- **JavaScript ES6+**: 主要编程语言
- **DOM 操作**: 原生 JavaScript DOM API
- **CSS 选择器**: 用于元素识别和样式处理

### 开发工具
- **Node.js**: 开发环境
- **ESLint**: 代码质量检查
- **自定义构建脚本**: 项目构建和打包

### 浏览器兼容性
- **Chrome**: 完全支持
- **Firefox**: 完全支持
- **Edge**: 完全支持
- **Safari**: 基本支持

### 用户脚本管理器
- **Tampermonkey**: 推荐使用
- **Greasemonkey**: Firefox 支持
- **Violentmonkey**: 跨浏览器支持

## 技术架构

### 模块化设计
- **translationCore.js**: 翻译核心逻辑
- **pageMonitor.js**: 页面监控和自动翻译
- **virtualDom.js**: 虚拟 DOM 操作
- **config.js**: 配置管理
- **utils.js**: 通用工具函数

### 性能优化
- 智能DOM监听
- 翻译缓存机制
- 延迟加载
- 资源压缩
