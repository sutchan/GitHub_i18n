## 项目概述

GitHub_i18n 是一个浏览器用户脚本，旨在为 GitHub 网站提供中文界面翻译功能。该插件通过智能识别页面元素，将英文界面翻译为中文，提升中文用户的使用体验。

## 核心特性

- 智能页面识别与翻译
- 支持多种 GitHub 页面类型
- 可配置翻译选项
- 自动更新机制
- 高性能 DOM 监控
