# 项目规范 (project.md)

## 项目概述
GitHub 中文本地化解决方案，包含用户脚本及自动化工具。

## 目录结构
- `src/`: 用户脚本源码
- `build/`: 构建逻辑及源码
- `dist/`: 构建产物 (忽略)
- `utils/`: 自动化字符串更新工具
- `openspec/`: 项目规范与报告
- `openspec/docs/`: 项目详细文档库 (原 `docs/`)
- `test/`: 调试与测试脚本

## 项目信息
- **项目 URL**: https://github.com/sutchan/GitHub_i18n
- **默认署名**: Sut
- **主语言**: JavaScript
- **目标平台**: 浏览器 (作为用户脚本运行)
- **开发环境**: Node.js
