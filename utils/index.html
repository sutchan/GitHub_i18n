GitHub 中文翻译插件 - 工具集

用于辅助 GitHub 中文翻译项目开发的实用工具集合
app.js
主应用程序入口，提供工具集的基本功能和交互界面。
使用方法：
node app.js
auto_string_updater.js
自动字符串更新工具，用于批量更新和管理翻译词典中的字符串。
使用方法：
node auto_string_updater.js [options]
cleanup.js
代码清理工具，用于格式化代码、移除冗余内容和修复常见问题。
使用方法：
node cleanup.js [file_path]
dictionary_processor.js
词典处理器，用于转换、合并和优化翻译词典文件。
使用方法：
node dictionary_processor.js [input_file] [output_file]
github_page_collector.js
GitHub页面收集器，用于抓取GitHub界面元素以便添加翻译。
使用方法：
node github_page_collector.js [url_pattern]
